# YANG Data Models for fine grain Optical Transport Network

This is the working area for the IETF [CCAMP Working Group](https://datatracker.ietf.org/wg/ccamp/documents/) Internet-Draft, "YANG Data Models for fine grain Optical Transport Network".

* [Editor's Copy](https://italobusi.github.io/draft-tan-ccamp-fgotn-yang/#go.draft-ietf-ccamp-fgotn-yang.html)

* [Working Group Draft](https://datatracker.ietf.org/doc/html/draft-ietf-ccamp-fgotn-yang)
* [Compare Editor's Copy to Working Group Draft](https://italobusi.github.io/draft-tan-ccamp-fgotn-yang/#go.draft-ietf-ccamp-fgotn-yang.diff)

## Command Line Usage

Formatted text and HTML versions of the draft can be built using `make`.

```sh
$ make
```

Command line usage requires that you have the necessary software installed.  See
[the instructions](https://github.com/martinthomson/i-d-template/blob/main/doc/SETUP.md).
